# Problem 1

Complete the following table

| Hex address | Dotted-decimal address |
| ----------- | ---------------------- |
| 0x0         | ???                    |
| 0xffffffff  | ???                    |
| 0x7f000001  | ???                    |
| ???         | 205.188.160.121        |
| ???         | 64.12.149.13           |
| ???         | 205.188.146.23         |

## Answer

| Hex address | Dotted-decimal address |
| ----------- | ---------------------- |
| 0x0         | 0.0.0.0                |
| 0xffffffff  | 255.255.255.255        |
| 0x7f000001  | 127.0.0.1              |
| 0xcdbca079  | 205.188.160.121        |
| 0x400c950d  | 64.12.149.13           |
| 0xcdbc9217  | 205.188.146.23         |
