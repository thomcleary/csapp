# Problem 2

Fill in the blank entries in the following table, giving the decimal and hexadecimal representations of different powers of 2:

| n   | 2^n (decimal) | 2^n (hexadecimal) |
| --- | ------------- | ----------------- |
| 9   | 512           | 0x200             |
| 19  | ???           | ???               |
| ??? | 16,384        | ???               |
| ??? | ???           | 0x10000           |
| 17  | ???           | ???               |
| ??? | 32            | ???               |
| ??? | ???           | 0x80              |

## Answer

| n    | 2^n (decimal) | 2^n (hexadecimal) |
| ---- | ------------- | ----------------- |
| 9    | 512           | 0x200             |
| 19   | `524,288`     | `0x80000`         |
| `14` | 16,384        | `0x4000`          |
| `16` | `65,536`      | 0x10000           |
| 17   | `131,072`     | `0x20000`         |
| `5`  | 32            | `0x20`            |
| `7`  | `128`         | 0x80              |
