# Problem 3

Given a 32-bit virtual address space and a 24-bit physical address, determine the number of bits in the VPN, VPO, PPN, and PPO for the following page sizes P:

| P   | VPN Bits | VPO bits | PPN bits | PPO bits |
| --- | -------- | -------- | -------- | -------- |
| 1KB | ???      | ???      | ???      | ???      |
| 2KB | ???      | ???      | ???      | ???      |
| 4KB | ???      | ???      | ???      | ???      |
| 8KB | ???      | ???      | ???      | ???      |

## Answer

| P   | VPN Bits | VPO bits | PPN bits | PPO bits |
| --- | -------- | -------- | -------- | -------- |
| 1KB | 22       | 10       | 14       | 10       |
| 2KB | 21       | 11       | 13       | 11       |
| 4KB | 20       | 12       | 12       | 12       |
| 8KB | 19       | 13       | 11       | 13       |
